Adding entities

Contracts, Orchards, Shipments, and Inspections can be added through the Tasks side bar. Simply click on the associated add_circle_outline,fill out the form, and click Create.


Progress

Tracking Progress: Overall progress, and total counts, can be tracked via the Progress side bar.
Progress of individual Contracts can be tracked via the contract table and contract details section.

Updating Progress: For convenience, a user can Complete (check) and/or Renew (autorenew) an Orchard via the associated icons in the contract table and contract details section. Complete matches the Orchard count to the contract amount while renew sets the Orchard count to zero. It is also possible to update the Orchard count to a specific number by editing the Orchard via the modal window.


Listing entities

Orchards, Shipments, and Inspections can be listed through the Tasks side bar. Simply click on the associated list; the list of entities will appear in a modal window. Clicking outside the menu, clicking the X button, or clicking the Cancel button will close the window.


Editing entities

Orchards: can be edited through the Contract table via the Edit Orchard icon (create).
Contracts: Similarly, Contracts can be edited through the Contract details section via the same icon (create).
